# Quick Fix - If You See Blank Screen

## Try This:

1. **Hard Refresh** the browser:
   - Press `Ctrl + Shift + R` (Windows)
   - Or `Ctrl + F5`

2. **Clear Browser Cache**:
   - Press `F12`
   - Right-click the refresh button
   - Select "Empty Cache and Hard Reload"

3. **Check Console for Errors**:
   - Press `F12`
   - Click "Console" tab
   - Look for red error messages
   - Take a screenshot and show me

4. **Try Incognito Mode**:
   - Press `Ctrl + Shift + N`
   - Go to http://localhost:3000
   - See if it works there

## If Still Blank:

The issue might be with the theme context. Let me know what error you see in the console and I'll fix it immediately!

## Expected Behavior:

You should see:
- Beautiful dashboard with gradient colors
- Sun/Moon icon for theme toggle
- "Add Company" button
- Your name in the top right

If you see NOTHING (blank screen), there's likely a JavaScript error.
